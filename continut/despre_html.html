Limbajul HTML

HTML(HyperTextMarkupLanguage) este un limbaj de marcare utilizat pentru a marca pagini web.

Un exemplu de cod html il gasiți în imaginea viitoare care se află la adresa https://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.
Imaginea are atributul height de valoare 147.

Tag-urile html cele mai utilizate în crearea tabelelor sunt:

table definește un tabel
tr un rând în tabel
td o coloana in tabel

Observatie: Pentru a pune un contur unui tabel se poate (dar nu se recomandă) să se puna atributul border="1".
La validare va da eroarea:The border attribute on the table element is absolete.

Formularele se pot crea cu ajutorul tag-ului form care poate conține tag-urile:input, textarea, label, select, option, button.
Pentru mai multe detalii consultați cursul sau tutorialul HTML Forms and Output de la adresa https://www.w3schools.com/html/html_forms.asp.

Pentru a crea liste se pot folosi tag-urile:

  1. ul-lista neordonată
  2. ol-lista ordonată
  3. li-item din listă
  4. dl-listă cu descrieri
  5. dt-termen dintr-o listă cu descrieri
  6. dd-descrierea dintr-o listă cu descrieri

Pentru a însera un document HTML în altul se poate folosi tag-ul < iframe > .
Astfel,exemplul de mai jos inserează conținutul de la adresa "https://html.spec.whatwg.org/multipage/".
